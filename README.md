# grayzone.nvim
Grayscale and red theme for neovim.

<img width="533" height="253" alt="Screenshot 2026-01-01 at 23 49 05" src="https://github.com/user-attachments/assets/112276e7-1a0f-421f-9403-e33d74434eeb" />

## Installing

1. Copy the `grayzone.lua` to your nvim's colors folder, typicaly in `~/.config/nvim/colors/`.
2. Add `vim.cmd.colorscheme("grayzone")` to your init.lua file.
